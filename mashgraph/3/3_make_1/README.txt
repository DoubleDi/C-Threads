Операционная система - linux
Сборка:
Из главной папки сборка производится утилитой make, командой makе или make all
Запуск:
Из главной папки запуск прозводится командой ./grass

Подразумевается, что система исправно работает с opengl 3.3. 
Иначе возможно, придется устанавливать совместимость с текущим opengl (не является ошибкой).
При написании в .bashrc было прописано export MESA_GL_VERSION_OVERRIDE="3.3COMPAT"
